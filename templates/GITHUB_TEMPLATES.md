## 🎯 Guide d'Utilisation

### Cloner un Template dans Projects

```bash
# 1. Aller dans projects/
cd /Users/lounis/Antigravity/Github/PersoDev/projects/

# 2. Cloner le template
git clone <url-github> mon-nouveau-projet

# 3. Renommer le remote (optionnel)
cd mon-nouveau-projet
git remote rename origin template
git remote add origin <mon-repo-perso>

# 4. Installer les dépendances
npm install  # ou pnpm install, ou pip install -r requirements.txt

# 5. Commencer à coder !
```

---

### Script de Clonage Rapide

Créer un script helper :

```bash
# templates/clone-template.sh
#!/bin/bash

TEMPLATE_TYPE=$1
PROJECT_NAME=$2

case $TEMPLATE_TYPE in
  "nextjs")
    git clone https://github.com/ixartz/Next-js-Boilerplate "$PROJECT_NAME"
    ;;
  "api-node")
    git clone https://github.com/hagopj13/node-express-boilerplate "$PROJECT_NAME"
    ;;
  "api-fastapi")
    git clone https://github.com/fastapi/full-stack-fastapi-template "$PROJECT_NAME"
    ;;
  "mobile")
    git clone https://github.com/obytes/react-native-template-obytes "$PROJECT_NAME"
    ;;
  "saas")
    git clone https://github.com/wasp-lang/open-saas "$PROJECT_NAME"
    ;;
  "ds")
    pip install cookiecutter-data-science && ccds "$PROJECT_NAME"
    ;;
  *)
    echo "Type inconnu. Utilisez: nextjs, api-node, api-fastapi, mobile, saas, ds"
    exit 1
    ;;
esac

echo "✅ Template $TEMPLATE_TYPE cloné dans $PROJECT_NAME"
```

**Usage:**
```bash
chmod +x templates/clone-template.sh
./templates/clone-template.sh saas mon-app-saas
```
